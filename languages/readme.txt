This archive contains Language pack for Stp Database Administrator
Visit the DBAdmin web site to find out more: http://www.stpworks.com

-----------------------------------------------
Installation:
-----------------------------------------------
Upload the *.lang files into "languages" folder of DBAdmin and select your language in Settings page
